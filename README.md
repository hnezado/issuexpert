# IssueXpert

Web application for incident and ticket management in a company environment.

## 🚀 Description

IssueXpert is a web-based system that allows users and technicians to manage support tickets efficiently.  
It includes user roles, ticket tracking, messaging between users, and future support for file attachments and notifications.

---

## 🧱 Tech Stack

- Node.js
- Express
- MySQL
- dotenv
- nodemon (dev)
